Docker in Docker
================


Collection of POCs for static and dynamic  load balancing, and running multiple Docker containers
in a single Docker image.

- static_proxy: a static proxy example using HAProxy.
- dynamic_proxy: a dynamic proxy example using skyDNS, skydock and a dynamic proxy
  written in python.
- dynamic_HAProxy: a dynamic proxy using HAProxy (TODO)
- docker_in_docker: a demonstration of a Docker image running other Docker images (TODO)
